// Title //
# Inferring-Product-Demand-from-Service-Level-Behaviour

// Overview //
This project explores whether service-level behaviour in UK hair salons can be used to evaluate product demand more effectively than trend-driven signals. Many haircare products are developed based on consumer trends (e.g. search intent and social media popularity). However, these signals do not always reflect how hair is actually styled and maintained in professional settings. This study investigates whether analysing services offered by stylists provides a more reliable foundation for product development and positioning decisions.

// Core Question //
Can service-level data be used to determine whether a haircare product should be launched or repositioned in the UK haircare market?

// Hypothesis //

Trend signals may suggest demand for certain products (e.g. curl creams), but products that are structurally required within high-frequency services represent more reliable, commercially grounded demand.

This study focuses on Knotless braid services across UK-based stylists, selected due to:

high frequency across providers
consistent service structure
clear dependency on specific product categories

// Approach //
Collect service data from a sample of UK stylists
Normalise and categorise services
Identify high-frequency service patterns
Map services to required product categories
Infer product demand from service dependencies

// Expected Outputs //
Service frequency analysis
Service to product dependency mapping
Product opportunity identification
Example product evaluation 
Example repositioning strategy for existing products
Commercial Application

This framework can be used to:

Evaluate whether a new product should be launched
Identify opportunities to reposition existing products
Understand which product categories are structurally embedded in real service workflows
